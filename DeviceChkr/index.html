Device Chkr v1.1

by Jay Versluis

 

A utility app for developers who need to know what type of device they're dealing with. This is important to determine the correct software download for a partuclar device and to check a device capabilities.

All iOS devices have an internal identifier (such as "iPhone6,1" or "iPad2,2") by which the exact model can be determined. This app shows that string as the Platform ID and translates it into a meaningful title, for example "This is an iPad 2 (GSM)".

This app also shows the Compile Time ID by which your code can check if it detects an iPhone, iPad or simulator.
